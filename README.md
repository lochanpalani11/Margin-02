# 🧮 Final Bid Calculator

Margin includes a built-in Final Bid Calculator that estimates the true cost of an auction purchase before you place a bid.

It automatically accounts for:

- Buyer's premium
- Sales tax
- Lot fees
- Additional auction fees

This helps ensure you know your actual investment before purchasing an item.

# 📲 Installation

The easiest way to use Margin is by downloading the latest APK.

1. Open the **Releases** section of this repository.
2. Download the latest **Margin.apk**.
3. Transfer it to your Android device if needed.
4. Allow installation from unknown sources if prompted.
5. Install the APK.
6. Open Margin and start tracking your inventory.

---

# 🏗️ Building from Source

## Requirements

- Android Studio
- Android SDK (API 26+)
- JDK 17

## Steps

```bash
git clone https://github.com/lochanpalani11/Margin.git

cd Margin
```

Open the project in Android Studio, allow Gradle to sync, and click **Run**.
